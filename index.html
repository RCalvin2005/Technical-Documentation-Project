Introduction

These notes are intended for personal use, and is by no means comprehensive. All documentation and further explanation of CSS properties can be found on W3Schools.

Box Model

Height and width can be altered using their respective properties using units such as pixels percentages and can be assigned a minimum and/or maximum value.

width: 80%; max-width: 600px;

Padding and margin can both be assigned individually or by using the shorthand property in the order of either top, right, bottom, left (clockwise order) or top-bottom, left-right. Setting the left-right margin to auto will center the element

padding-top: 10px; margin: 5px auto;

Padding and borders would typically increase the width and height of the element. To avoid this, the following property can be used:

box-sizing: border-box;

A border can be added using its shorthand property which sets the value of width, style and color in order. The following are some common border styles:

  • Solid
  • Double
  • Dashed
  • Dotted

A border can also be rounded by using the border-radius shorthand property.

border-radius: 5px;

You can set the background color of an element using hexadecimal, rgba or hsla formats. Just make sure there is enough contrast for the text to be legible.

background-color: rgba(23, 54, 33, 0.8);
Typography

The font used can be set using the font-family property. Make sure to set a backup font to be safe.

font-family: 'Open Sans', sans-serif;

Font size can be changed using the font-size property. Pixels is a common units but for responsive designs, using em or rem would be better.

font-size: 2rem;

The color property assigns the color of text. You can use hexadecimal, rgba or hsla.

color: #D3D3D3;

To change the boldness of the font, use font-weight. You could set it to bold or assign a numeric value.

font-weight: 800;

To make a text italic, use the font-style property. It can also be set to oblique.

font-style: italic;

Use the text-align property to align text to the left, right, center, or justify (equal width) within its element.

text-align: justify;

You can make text more legible by increasing the spacing between lines using line-height. Can also be used to vertically center a text when set equal to the element height.

line-height: 1.4;

To make fancier titles, you can use the letter-spacing property.

letter-spacing: 2px;
Displays and Position

The you can set the position property to static, absolute, fixed, relative and sticky. Then you can move the element using the top, bottom, left, and right properties.

position: fixed; top: 0; right: 0;

You can add depth to your website using the box-shadow property which takes the arguments horizontal-offset, vertical-offset, blur, spread and color.

box-shadow: 5px 5px 10px #115020;

For images with a container, you should typically use the object-fit property that is set to contain or cover.

object-fit: cover;

When the content goes beyond the element, you could set the overflow to hidden or auto to add a scroll-bar.

overflow: hidden;

You can change the cursor style to better highlight certain aspects such as links.

cursor: pointer;

You can add transitions by setting the transition-property and transition-duration in combination with a pseudo-selector like hover.

transition: all 0.8s;

The following are common display behaviors that'll be useful in styling certain elements:

  • None
  • Block
  • Inline
  • Flex
  • Grid
Flexbox

Flexbox makes responsive design easier by making the display flexible. To use Flexbox, set the display of the container (not the items themselves) to flex.

You can set the flex-direction property to row, column, row-reverse or column-reverse.

To avoid the container from going off screen when the viewport gets too small, set the flex-wrap to wrap or wrap-reverse.

Or you could directly use the flex-flow shorthand to set both the flex-direction and flex-wrap.

The justify-content property is used to align items along the flex-direction and can be set to center, flex-start, flex-end, space-around and space-between.

In contrast, the align-items property is used to align items along the cross-axis and can be set to center, flex-start, flex-end, stretch or baseline

The align-content property is similar to align-items but it aligns the flex line rather than the items themselves and can be set to space-between, space-around, stretch, center, flex-start and flex-end.
